# react-native-section-indexer-kor
fastscroll 혹은 section indexer 라고 부르는 초성으로 리스트 좌표로 이동 하는 스크롤뷰 입니다.

## Getting started

`$ npm install react-native-section-indexer-kor --save`

### 사용 방법
example 확인

### 기본값

textSize = 12

textColor = '#8a8a8a'

divSize = 5

divColor = '#b3b3b3'

indexerWidth = 20

indexerColor = '#f2f2f2'

indexerBorderR = 10

indexerRightMargin = 5
